# Brand Colors

This document outlines the brand color palette for Magic Worlds.

## Primary Colors

### Primary - Teal/Cyan

- **Hex**: `#40b0bf`
- **Usage**: Primary brand color used in gradients, accents, and key UI elements
- **Description**: A vibrant teal/cyan color that represents the core brand identity

### Secondary - Golden Yellow/Mustard

- **Hex**: `#d2a64e`
- **Usage**: Secondary brand color used in gradients and complementary elements
- **Description**: A warm, muted golden yellow/mustard shade that adds warmth to the palette

### Accent - Green

- **Hex**: `#04d27f`
- **Usage**: Accent color used in gradients and highlight elements
- **Description**: A fresh, vibrant green that provides energy and growth symbolism

## Gradient Usage

The brand colors are typically used together in gradients following this pattern:

```
from-[#40b0bf] via-[#d2a64e] to-[#04d27f]
```

This creates a smooth transition from teal/cyan through golden yellow to green, representing the brand's dynamic and vibrant nature.

## Implementation

- Colors are defined in `tailwind.config.ts` under the `brand` theme extension
- Used throughout the application in:
  - Hero sections and banners
  - Card components and stat displays
  - Chart visualizations
  - Interactive elements and hover states
  - Navigation and footer links
  - Timeline and event markers

## Color Palette Summary

| Color     | Hex Code  | Role             |
| --------- | --------- | ---------------- |
| Primary   | `#40b0bf` | Main brand color |

## Notes

- White and grey colors remain unchanged as per design requirements
- All gradients and brand-colored elements have been updated to use this palette
- Colors are accessible and maintain good contrast ratios for readability
